Web App: Platform

Description

The entire platform used by customers to order satellite imagery as well as view image archives

Sample usage:

  • Open homepage: loads a map
  • Select shape from toolbar at top: rectangle or polygon
  • Click on map to overlay the shape. Click and drag to adjust size of area to image
  • Right click on shape and choose options such as "still or video", "frame rate", "time of day desired", etc. from modal box
  • Draw multiple shapes
  • Click submit when done
  • Algorithm checks if this area has been imaged before, and shows a thumbnail if it has. User has option to buy this archived image

Features

  • User authentication

Tech stack

  • Frontend: Bootstrap, Google Maps API
  • Backend: Django in Python, AJAX, MySQL db
